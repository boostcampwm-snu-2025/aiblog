# 학습 목표
* 다양한 외부 API를 연동해서 서비스를 구현할 수 있다.
* LLM을 연동한 AI 서비스를 만들 수 있다.
* AI를 활용해서 애플리케이션을 설계하고 구현할 수 있다.

# 기능요구사항
## 3주차 미션 : LLM 연동
* 생성된 글 저장/관리
* 저장된 글 목록/상세 보기 구현
* 기타 미 구현된 기능개발

# 프로그래밍 요구사항
## 서비스 코드 다듬기 (리팩토링)
* 생성된 글 저장/관리는 localStorage기반의 클라이언트를 기본으로 한다.
* 전역 상태 관리(Context API + useReducer) 도입 localStorage 동기화 작업을 한다.
* 비동기 상태 패턴 적용(idle/loading/success/error)과 UI 반영
* 중복 로직 커스텀 훅 분리시도
* 기타 리팩토링 작업
